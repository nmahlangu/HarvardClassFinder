[ Welcome to Theharvardclassfinder ]

Theharvardclassfinder is a database that makes choosing classes at Harvard easier.



You can use Theharvardclassfinder to:

  • Discover new classes
  • See how classes stack up against each other in any, or all, selected department(s)
  • See which classes have been rated the highest by your peers
  • Find classes that allow you to balance academics and extracurriculars
  • Rank classes by course #, overall Q score, difficulty, workload, and peer recommendations
